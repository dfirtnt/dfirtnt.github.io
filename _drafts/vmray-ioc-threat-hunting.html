This blog entry walks through setting up a Splunk dashboard which marries VMRay IOC data with live telemetry to identify any matches which can be investigated as possible compromises.

We will cover the following:

  1. Setting up the VMRay App for Splunk

  2. Exploring the VMRay data structure

  3. Using subsearches to find matches

  4. Creating custom Splunk dashboards

VMRay is one of the leading commercial tools for detonating malware and collecting intelligence on how the malware operates. Their API is pretty robust, but luck for us, VMRay also supports a Splunk app which is available in the Splunkbase repository.
